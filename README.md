# Promoção Dia das Mães - Buffet Biruta Park

## Rodar localmente
```bash
npm install
npm start
```

- Site: `http://localhost:3000`
- Admin: `http://localhost:3000/admin`

## Arquivos úteis
- Logo flutuando: `public/assets/logo-biruta.png`
- Imagem principal do cadastro: `public/assets/cadastro-banner.jpg`
- Banco local: `data/participants.json`

## Login admin
- usuário: `biruta`
- senha: `biruta@2026`
